contextualize-js

A context-aware feature flag system for JavaScript, enabling features based on user roles, location, and time.

Installation

npm install contextualize-js

Usage Example


import FeatureFlag from 'contextualize-js';

const featureFlag = new FeatureFlag();

// Enable a feature with a role and time context
featureFlag.enable('dark-mode', { role: 'admin', time: '2024-12-31T23:59:59Z' });

console.log('Is dark mode enabled?', featureFlag.isEnabled('dark-mode')); // true

// Disable the feature
featureFlag.disable('dark-mode');
console.log('Is dark mode enabled?', featureFlag.isEnabled('dark-mode')); // false

API Documentation

  • enable(flag: string, context: FlagContext): void – Enables a feature with context.
  • disable(flag: string): void – Disables the specified feature.
  • isEnabled(flag: string): boolean – Checks if the feature is enabled.
